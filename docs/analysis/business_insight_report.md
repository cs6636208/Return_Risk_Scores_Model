# 📊 Business Insight Report: Return & Refund Risk Analysis (V3)
**Project:** GMM O Shopping - Return Risk Prediction System
**Phase:** Week 3 - Exploratory Data Analysis (Final Summary)

---

## 🎯 Executive Summary
จากการวิเคราะห์ข้อมูลออเดอร์ย้อนหลัง 5,000 รายการ เราพบว่าอัตราการคืนสินค้าไม่ได้เกิดขึ้นแบบสุ่ม แต่มี **"Pattern ที่ชัดเจน"** ซึ่งสัมพันธ์กับช่องทางขาย, ประเภทสินค้า และพฤติกรรมการซื้อที่สามารถทำนายได้ล่วงหน้า Insight เหล่านี้จะถูกเปลี่ยนเป็น AI Features เพื่อลดต้นทุนความล้มเหลว (Cost of Return) ของบริษัท

---

## 🔍 5 Key Strategic Insights (การค้นพบที่สำคัญ)

### 1. The "High-Risk Channel-Category" Combo
*   **Insight:** สินค้ากลุ่ม **Fashion** ที่ขายผ่านช่องทาง **TV_Show** มีอัตราการคืนสูงสุดที่ **~18-20%**
*   **Root Cause:** "Expectation Gap" ลูกค้าคาดหวังความสวยงามตามที่เห็นนางแบบในรายการทีวี เมื่อได้รับของจริงแล้วไม่ตรงปกหรือไซส์ไม่ได้ จึงเกิดการคืนสูงกว่าปกติ
*   **Business Action:** เพิ่มขั้นตอน "Size Confirmation call" ทันทีสำหรับออเดอร์ Fashion จาก TV Show

### 2. Payment Method Sensitivity (COD Risk)
*   **Insight:** ออเดอร์แบบ **เก็บเงินปลายทาง (COD)** มีโอกาสถูกคืน (หรือปฏิเสธการรับ) สูงกว่าการจ่ายเงินก่อนถึง **2.5 เท่า**
*   **Root Cause:** "Low Commitment" ลูกค้าไม่มีความเสี่ยงทางการเงินในการสั่งซื้อ ทำให้ตัดสินใจคืนได้ง่ายกว่า
*   **Business Action:** พิจารณาระบบสะสมแต้มพิเศษสำหรับลูกค้าที่เลือกจ่ายเงินทันที (Non-COD) เพื่อลดความเสี่ยง

### 3. Logistics "Tipping Point"
*   **Insight:** เมื่อการจัดส่งล่าช้าเกิน **+1 วัน** จากที่สัญญาไว้ อัตราการคืนจะพุ่งขึ้นจาก 5% เป็น **15% ทันที**
*   **Root Cause:** "Emotional Cooling" อารมณ์อยากได้สินค้าลดลงตามเวลาที่รอนานขึ้น
*   **Business Action:** ระบบต้อง Alert ทีมขนส่งเมื่อออเดอร์เข้าสู่วันที่ต้องส่ง (Target Date) เพื่อเร่งจัดส่งกลุ่มเสี่ยงก่อน

### 4. Bracketing Behavior Detection (Buy-to-Try)
*   **Insight:** ออเดอร์ที่มีการสั่งสินค้าชนิดเดียวกัน **มากกว่า 1 ชิ้น** (เช่น เสื้อ 2 ตัวคนละสี) มักจบลงด้วยการคืนสินค้า 1 ชิ้นเสมอ
*   **Root Cause:** พฤติกรรมการ "สั่งมาลอง" เพื่อเลือกตัวที่ถูกใจที่สุด
*   **Business Action:** ปรับนโยบายการคืนสินค้าสำหรับกลุ่ม Bracketing หรือนำเสนอ Virtual Fitting Room เพื่อช่วยตัดสินใจ

### 5. High-Discount Impulse Buying
*   **Insight:** ออเดอร์ที่มีส่วนลดมากกว่า **20%** มีอัตราการคืนสูงกว่าปกติ
*   **Root Cause:** ลูกค้าสั่งซื้อเพราะ "ราคา" ไม่ใช่เพราะ "ความต้องการ" จริงๆ (Impulsive Buy)
*   **Business Action:** ตรวจสอบคุณภาพสินค้ากลุ่ม Flash Sale เป็นพิเศษก่อนส่ง เพื่อไม่ให้มีข้ออ้างในการคืน

## 💡 Actionable Recommendations (ข้อเสนอแนะเชิงกลยุทธ์)

### 🟢 สำหรับฝ่ายปฏิบัติการ (Operations/Call Center):
1.  **AI-Driven Alert:** ติดตั้งระบบแจ้งเตือนสีแดงในหน้าจอพนักงาน เมื่อออเดอร์เข้าข่าย "High Risk" (เช่น TV + Fashion + COD)
2.  **Proactive CRM:** ในกรณีที่จัดส่งล่าช้า ให้ส่ง SMS แจ้งสิทธิพิเศษหรือส่วนลดเล็กน้อยเพื่อ "ซื้อเวลา" และรักษาความรู้สึกดีของลูกค้า

### 🔵 สำหรับฝ่ายการตลาดและจัดซื้อ (Marketing/Merchandising):
1.  **Product Quality Audit:** ตรวจสอบแบรนด์ที่มีคะแนนรีวิวต่ำกว่า 4.0 ดาวอย่างเข้มงวด เพราะสัมพันธ์โดยตรงกับการคืนเนื่องจาก "สินค้าชำรุด"
2.  **Tier-Based Policy:** มอบสิทธิพิเศษการคืนสินค้าฟรีให้เฉพาะกลุ่ม Gold/Platinum เพื่อลดต้นทุนการจัดการในกลุ่มลูกค้าขาจร
