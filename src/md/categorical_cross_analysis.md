# 📊 รายงานผลการวิเคราะห์ Categorical Cross Features กับอัตราการคืนสินค้า

เนื่องจากตัวแปรประเภทข้อความ (Categorical) ไม่สามารถคำนวณค่าสหสัมพันธ์ (Correlation) ร่วมกับตัวเลขได้ตรงๆ การวิเคราะห์ที่ดีที่สุดคือการนำตัวแปรข้อความมา **Cross (ทำตารางไขว้)** แล้วคำนวณ **อัตราการส่งคืนจริง (Mean Return Rate)** ซึ่งผลลัพธ์จะเป็น **ตัวเลข (สัดส่วน 0 ถึง 1)** ทำให้เราเห็นภาพความสัมพันธ์และสามารถเปรียบเทียบความเสี่ยงได้อย่างชัดเจนครับ
